# סידור חכם לכיתה

אפליקציית web למורים לבניית סידור הושבה חכם לכיתה.

## הרצה מקומית

```bash
npm install
npm run dev
```

עמוד הפיתוח עולה על http://localhost:5173

## Stack

Vite + React + TypeScript + Tailwind CSS, עם react-konva לקנבס, zustand ל-state, ו-Supabase ל-sync ענן (אופציונלי).
